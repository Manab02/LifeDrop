# 🩸 LifeDrop

LifeDrop is a MERN Stack based Blood Donation Management System that connects **Donors, Hospitals, Organizations, and Recipients** on a single platform. It helps users find blood donors based on blood group and location, manage blood inventory, and make the blood donation process faster and more efficient.

---

## 🚀 Features

### 👤 User Authentication
- Secure Login & Registration
- JWT Authentication
- Role-Based Access Control

### 🩸 Donor
- Register as a blood donor
- Update donor profile
- Search nearby donors
- View donation history

### 🏥 Hospital
- Manage blood inventory
- Request blood
- View donor information
- Manage blood transactions

### 🏢 Organization
- Organize blood donation camps
- Manage donors
- Track blood collection

### 🔍 Search & Location
- Search donors by blood group
- Search hospitals and organizations
- Find nearby donors using location-based search

### 📊 Dashboard
- Blood inventory statistics
- Total donors
- Blood request history
- Transaction records

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Manab02/LifeDrop.git
```

Move into project folder

```bash
cd LifeDrop
```

Install Backend Dependencies

```bash
npm install
```

Install Frontend Dependencies

```bash
cd client
npm install
```

---

# ▶️ Run the Project

Start Backend

```bash
npm run server
```

Start Frontend

```bash
cd client
npm start
```

---

# 🌐 Environment Variables

Create a `.env` file in the root directory.

PORT=5000

MONGO_URL=your_mongodb_connection_string

JWT_SECRET=your_secret_key
